Multiple SeqDoC

Multiple SeqDoC uses the same algorithm as SeqDoC, but allows the comparison of up to ten test sequences against a single reference sequence simultaneously. This gives minor performance benefits (up to about 25% speed improvement), and simplifies analysis. Otherwise use is the same as for the standard SeqDoC program.

Users with larger numbers of sequences are strongly encouraged to investigate using the Staden package, which is better suited (and much faster) for SNP identification in multiple sequences.

Please be aware that all alignments returned will be truncated to the length of the shortest trace (this is also the case for regular SeqDoC, but can have more significant implications for multiple SeqDoC). Also, please note that despite the speed benefits of multiple alignments, this program will probably still take between 10s and 15s per test sequences (i.e. maybe more than two minutes for ten samples). Please be patient and don't hit 'reload' - that will only slow things down more.

Source code for Multiple SeqDoC is available here. System requirements and accessory applications are the same as for regular SeqDoC.
